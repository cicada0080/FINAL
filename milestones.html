Основа

Основа сайта: HTML (HyperText Markup Language) HTML — это «скелет» страницы. Он определяет структуру и содержание: где заголовок, где текст, где картинки и ссылки.

Оформление: CSS (Cascading Style Sheets) CSS — это «одежда» сайта. Он отвечает за внешний вид: цвета, шрифты, размеры, отступы и расположение элементов.Без CSS все сайты выглядели бы как простой текст на белом фоне

Интерактивность: JavaScript (JS) JavaScript — это «мозг» сайта. Он отвечает за поведение страницы, которая реагирует на действия пользователя без перезагрузки. Что умеет JavaScript Показывать всплывающие окна (alert, confirm). Менять содержимое страницы «на лету». Проверять правильность заполнения форм (например, email). Создавать слайдеры, анимации, сложные меню. Отправлять данные на сервер без перезагрузки (AJAX).
